如何搭建我的 GitHub Pages 个人网站

记录从仓库命名、页面设计、SSH 推送到公网访问的第一版搭建过程。

GitHub PagesAstro个人网站

目标

这个网站先承担两个职责：第一，作为公开的自我介绍入口；第二，作为技术博客、产品原型和思考记录的目录。

仓库命名

GitHub 用户主页需要使用 WangAo-0.github.io 作为仓库名。这样部署完成后，公网地址就是 https://wangao-0.github.io/。

第一版结构

  1. 首页：介绍我是谁、这个网站记录什么。
  2. 技术博客：放搭建记录、项目复盘和技术笔记。
  3. 技术 Roadmap：记录学习路径和阶段重点。
  4. 产品原型：记录值得长期推进的产品假设。
  5. 联系方式：保留邮箱、微信、资助二维码和自媒体入口。

后续迭代

后续我会逐步补充真实文章，并把每次项目实践中的关键决策、问题排查和复盘写下来。这样网站就不是静态名片，而是持续增长的知识库。
