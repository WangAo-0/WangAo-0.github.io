# WangAo-0 Personal Website

这是 `WangAo-0` 的 GitHub Pages 个人网站。第一版定位为个人知识主页，重点放在自我介绍、技术博客和思考记录。

公网地址：

```text
https://WangAo-0.github.io/
```

## 内容结构

- `index.html`：首页，包含自我介绍、技术博客入口、思考札记和联系方式。
- `blog/index.html`：博客目录。
- `blog/github-pages-site.html`：GitHub Pages 搭建记录示例文章。
- `blog/thinking-in-public.html`：公开写技术笔记的思考示例文章。
- `styles.css`：全站样式。
- `script.js`：导航交互、年份显示和首页 Canvas 视觉。

## 发布方式

仓库名应为 `WangAo-0.github.io`。GitHub Pages 设置中选择：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

## 后续维护建议

- 新文章可以复制现有博客文章页面，再修改标题、日期和正文。
- 首页的“技术博客”区块建议只放 3 到 6 篇最值得读的文章。
- 联系邮箱已设置为 `oliverdebox@163.com`。
- 如果之后文章数量变多，可以升级到 Astro、VitePress 或 Jekyll，用 Markdown 管理文章。
