Product Prototypes

产品原型

这里不是简单列想法，而是把长期值得做的方向拆成可验证、可复盘、可逐步实现的产品假设。

Agentic Developer Kit

自动化开发提效的 Agent 程序员包

一个不只是会写代码的 agent，而是包含外围环境、开发准则、测试规范和上线维护流程的完整产品交付系统。

  • 需求沟通
  • 方案设计
  • 开发实现
  • 测试验证
  • 上线发布
  • 维护优化
  • 新需求迭代

Finance Agent Platform

炒股理财的 Agent 平台

面向个人投资研究的 agent 平台，聚焦信息汇总、风险提示、策略复盘和决策辅助，不替代用户判断。

  • 资讯聚合
  • 组合跟踪
  • 风险提示
  • 策略复盘

Personal Insight System

个性化信息洞察系统

持续搜集、组织和总结个人关注的信息，并输出更有价值的 insight，而不是只做信息搬运。

  • 信息搜集
  • 主题聚类
  • 摘要生成
  • 洞察输出

Multi-Agent Collaboration

多 Agent 协作平台

探索多个 agent 如何分工、协作、互相审查和交付复杂任务，重点关注协议、边界和可观测性。

  • 任务拆解
  • 角色分工
  • 协作协议
  • 结果评审
